## How to play
- **Mouse** - Turn
- `W` `A` `S` `D` - Move
- **Wheel** - Rotate Torch
- **Space** - Jump
- **Shift(held)** - Run
- **C** - Crouch
- **Q** - Pick Objects
- **V** - Peek
- **E** - Drop Objects
- **R** - Rotate Object
- **K** - Kick Object
- **P** - Pause the Game
